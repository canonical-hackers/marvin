# Marvin #
Because we all hate those dammed doors.

## Requirements ##
* Ruby 1.9.2+
* Gems in the Gemfile

## Starting up ##
1. Install bundler if you have not
  `gem install bundler`
2. Install the gems needed
  `bundle install` 
3. Run the `auto_restart.sh` script in the root.
